# My User App

React + Node + MongoDB based user management app.

## Features
- User registration form
- Node.js + Express backend  
- MongoDB database
- React frontend

## Setup
1. Clone: `git clone https://github.com/Kanagashree/my-user-app.git`
2. Backend: `cd backend && npm install && node index.js`
3. Frontend: `cd frontend && npm install && npm start`

## Tech Stack
- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB
